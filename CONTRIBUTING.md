# 贡献指南

感谢你对本项目的关注！我们欢迎所有形式的贡献。

## 如何贡献

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的改动 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个 Pull Request

## 代码规范

- 保持代码简洁、可读
- 添加必要的注释
- 确保所有测试通过
- 遵循项目现有的代码风格

## 报告问题

如果你发现 bug 或有功能建议，请通过 [Issues](../../issues) 提交。

## 行为准则

请保持友好、专业的交流态度。
